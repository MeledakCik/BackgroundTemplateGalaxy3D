Galaxy: Interactive 3D Landing Page

Selamat datang di Galaxy, sebuah template landing page futuristik yang dibangun dengan Next.js 16 dan React. Template ini dirancang untuk memberikan pengalaman visual yang mendalam dengan estetika glassmorphism dan interaksi 3D yang halus.

🚀 Fitur Utama
- Next.js 16 & App Router: Arsitektur modern untuk performa maksimal.
- 3D Interactivity: Animasi responsif yang bergerak mengikuti kursor pengguna.
- Futuristic Design: Palet warna navy dan abu-abu dengan mesh gradient yang elegan.
- Optimized Performance: Dibuat dengan next/font untuk waktu loading yang cepat.
- Clean & Modular: Struktur kode yang rapi, mudah dipahami, dan siap untuk dikustomisasi.

🛠️ Cara Instalasi
Pastikan Anda sudah menginstal Node.js di komputer Anda. Ikuti langkah-langkah di bawah ini:

Clone repositori ini:

Bash
git clone (https://github.com/MeledakCik/BackgroundTemplateGalaxy3D.git)
cd BackgroundTemplateGalaxy3D
Instal dependensi:

Bash
npm install
# atau
yarn install
Jalankan server pengembangan:

Bash
npm run dev
Buka di Browser:
Buka http://localhost:3000 untuk melihat hasilnya secara langsung.

🎨 Kustomisasi
- Konten Halaman: Ubah teks, gambar, dan elemen utama pada file app/page.tsx.
- Styling: Sesuaikan warna dan tema melalui file konfigurasi Tailwind CSS Anda (tailwind.config.ts).
- Font: Kami menggunakan font Geist. Anda dapat mengubahnya di bagian app/layout.tsx.

📦 Deployment
Cara termudah untuk mempublikasikan proyek ini adalah melalui Vercel.
Hubungkan repositori Anda ke Vercel.
Klik Deploy.
Proyek Anda akan langsung aktif secara otomatis!
